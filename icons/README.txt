아이콘 파일 안내
================

필요한 아이콘 파일:
- icon-192x192.png (192x192 픽셀)
- icon-512x512.png (512x512 픽셀)

아이콘 생성 방법:
1. https://www.favicon-generator.org/ 방문
2. 원하는 이미지 업로드 또는 텍스트로 아이콘 생성
3. 192x192, 512x512 크기로 다운로드
4. 이 폴더에 저장

또는:
- Canva, Figma 등의 디자인 툴 사용
- "my note" 텍스트 또는 노트 아이콘 디자인

임시로 다음 사이트에서 빠르게 생성 가능:
- https://realfavicongenerator.net/
- https://favicon.io/

디자인 컨셉:
- 베이지 톤 (#f5f4f0) 배경
- 노트/메모장을 상징하는 심플한 디자인
- 둥근 모서리로 부드러운 느낌
